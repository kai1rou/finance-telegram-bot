# Финансовый Telegram бот

Бот для учета личных финансов на Python с использованием aiogram и SQLite.

## Функции
- Добавление расходов и доходов по категориям
- Просмотр отчетов за разные периоды
- Статистика по категориям
- Удобное меню с кнопками

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с токеном: `BOT_TOKEN=ваш_токен`
4. Запустить: `python bot.py`
